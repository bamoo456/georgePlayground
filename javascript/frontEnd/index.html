Nginx static file on nodeJs

nginx 的強大在於它的運作模式是類似於nodejs的event-driven；基於這樣的特色，加上用c語言實作的高效；讓nginx在處理request 時能夠更快速且不阻塞目前的process

why nginx for nodejs ?

事實上，nodejs的http module已經實作的非常完整，即使不靠nginx也能夠運作正常；然而nodejs的強項在於處理i/o 類的request，而身為web server又必須壓縮網頁內容，建立https的secure連線，以及讀取並轉送大量的靜態檔…

這些都必須大量使用到cpu 資源的，所以對nodejs而言反而是個負擔；然而這些tasks 對nginx而言卻是小菜一碟，用c語言實作的nginx剛剛好…
